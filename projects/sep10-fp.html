Freedom Project

Context

For context, this project is basically about everything we learned so far in the year (HTML, CSS, Github, and Bootstrap) into a major project that we've been technically working on for the entire year, the Freedom Project.

Content

We basically made a website about a topic of our choosing, the technologies (and possible future technologies of it), and presented all we learned so far into the website itself, as we are almost making it off from scratch. We started from the beginning of school by researching our topics, and I chose art as a topic, because it's something I'm passionate about. We also had to learn a tool to add to our website as well, and I chose jQuery because I felt like that fit my website the most. When we actually made our website near the end of the year, I decided to add a navbar and some cards using Bootstrap to help me, and I used jQuery for the last part of the website. Overall, I really liked how the website turned out at the end, and I'm quite proud of my project.
